Тестовое задание в компанию Faceter на позицию стажера
=============================
Vue.js + Typescript + ESLint + Bootstrap grid
https://faceter-yasha.netlify.app

## Установка проекта
```
npm install
```

### Компиляция в режиме разработки
```
npm run serve
```

### Компиляция и минификация
```
npm run build
```
